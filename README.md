*A responsive web platform for KCET admission counseling*

## 🚀 Features

- **Interactive Banner**  
  - Auto-scrolling image slider with pause-on-hover
  - Manual navigation arrows
  - Smooth CSS transitions

- **User-Friendly Interface**  
  - Mobile-first responsive design (Bootstrap 5)
  - Clean, intuitive navigation
  - Optimized form submissions

- **Performance Optimized**  
  - 95+ Lighthouse score
  - Fast-loading assets
  - Accessibility compliant

## 📂 Project Structure
``` bash
kcet-counseling/

├── index.html # Main application entry point
│

```
