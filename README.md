## 🚀 Getting Started

### Prerequisites
- Modern web browser (Chrome/Firefox/Edge)
- Git (optional)

### Installation
```bash
git clone https://github.com/Santhosh-Kumar-R-S/Vidyarthi-Academy.git
cd kcet-counseling
```

## Development
1. Open `index.html` in browser

2. For live editing:

  ```bash
  npx serve ./
```
